# Money X - Business Plan

## 1. Business Model

**Company:** Money X  
**Mission:** AI-powered financial automation for solopreneurs and small businesses

**Revenue Model:** SaaS Subscription (Freemium)
- Free tier: Basic features
- Pro ($19/mo): Advanced automation, unlimited usage
- Enterprise ($49/mo): Custom integrations, priority support

**Target Market:** Solopreneurs, freelancers, small agencies (1-10 employees)

---

## 2. Product: CashFlow AI

**Core Value Proposition:** Automate invoicing, expense tracking, and financial reporting with AI

**Features:**
1. **AI Invoice Generator** - Create professional invoices from conversation
2. **Expense Tracker** - Snap photos, AI categorizes and tracks
3. **Financial Dashboard** - Real-time revenue, expenses, profit metrics
4. **Recurring Payments** - Subscription management
5. **Client Portal** - Clients view/pay invoices online

**Technical Stack:**
- Frontend: HTML/CSS/JS (lightweight, fast)
- Backend: Node.js with Express
- Database: SQLite (for MVP), designed for Supabase migration
- Auth: JWT tokens
- Payments: Stripe-ready integration points

---

## 3. Technical Infrastructure

**Architecture:**
- RESTful API
- SPA frontend
- Local file storage (SQLite)
- Modular design for cloud deployment

**File Structure:**
```

/public/ - Static frontend
```
